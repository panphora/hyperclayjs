Parameters

NameTypeDefaultDescription

Returns

void

Example

toast('Something went wrong', 'error');

What is Hyperclay?

Hyperclay turns a single HTML file into a living app — one that can rewrite itself, persist changes, and run anywhere.

One HTML file, full application.

Callout Box (orange variant)

What is HyperclayJS?

HyperclayJS is the JavaScript library that powers single-file HTML apps. It gives your HTML files the ability to send themselves to a save endpoint and manage edit mode.
